Getting Started

O que é CapivaraJS?

Hoje temos vários frameworks que te possibilitam criar componentes, porém seus componentes ficam isolados naquela tecnologia, isso faz com que uma possível migração seja mais complexa.

O CapivaraJS é um framework híbrido e suas diretivas são totalmente personalizáveis, seus componentes funcionam independentemente da tecnologia utilizada.



Instalação

Para instalar a versão mais recente usando a ferramenta de linha de comando npm, execute o seguinte:

$  npm install --save capivarajs
