API-Reference

Funções

Criamos algumas funções que podem facilitar no seu desenvolvimento, abaixo vamos entender cada uma delas.

capivara.isArray

Determina se uma referência é um array, informe o valor a ser verificado por parâmetro. Exemplo: 

let pessoas = [{ nome: 'João' }];
let resultado = capivara.isArray(pessoas);

console.log(resultado); // O resultado deve ser TRUE.

capivara.isObject

Determina se uma referência é um objeto, informe o valor a ser verificado por parâmetro. Exemplo: 

let pessoa = { nome: 'Maria' };
let resultado = capivara.isObject(pessoa);

console.log(resultado); // O resultado deve ser TRUE.

capivara.isDate

Determina se uma referência é uma data, informe o valor a ser verificado por parâmetro. Exemplo: 

let pessoa = {
    nome: 'Maria',
    aniversario: new Date('1996-11-13T23:00:00Z')
};
let resultado = capivara.isDate(pessoa.aniversario);

console.log(resultado); // O resultado deve ser TRUE.

capivara.isElement

Determina se uma referência é um NodeElement, informe o valor a ser verificado por parâmetro. Exemplo: 

let resultado = capivara.isElement(document.body);
        
console.log(resultado); // O resultado deve ser TRUE.

capivara.isFunction

Determina se uma referência é uma função, informe o valor a ser verificado por parâmetro. Exemplo: 

let minhaFuncao = function(){
    //code
};
let resultado = capivara.isFunction(minhaFuncao);

console.log(resultado); // O resultado deve ser TRUE.

capivara.isNumber

Determina se uma referência é um número, informe o valor a ser verificado por parâmetro. Exemplo: 

let notaDoCapivara = 10;
let resultado = capivara.isNumber(notaDoCapivara);

console.log(resultado); // O resultado deve ser TRUE.

capivara.isString

Determina se uma referência é uma string, informe o valor a ser verificado por parâmetro. Exemplo: 

let nome = 'CapivaraJS';
let resultado = capivara.isString(nome);

console.log(resultado); // O resultado deve ser TRUE.

capivara.merge

Faça a mesclagem de objetos em um único objeto. Informe os parâmetros que serão mesclados como no exemplo abaixo: 

let obj1 = {
    nome: 'Maria'
};
let obj2 = {
    idade: 25
};
let resultado = capivara.merge(obj1, obj2);

console.log(resultado); // O resultado deve ser {nome: "Maria", idade: 25}.

capivara.copy

Você pode fazer a cópia de um objeto para que seja criado uma nova referência. Exemplo: 

let obj = {
    nome: 'Maria'
};
let resultado = capivara.copy(obj);

console.log(resultado);
// O resultado deve ser um objeto idêntico, entretanto com outra referência: {nome: 'Maria'}.

capivara.replaceAll

Você pode fazer o replace em uma String de todas as ocorrências encontradas. Exemplo: 

let str = 'Eu amo capivara. Capivara é demais.';
let resultado = capivara.replaceAll(str, 'capivara', 'CapivaraJS');

console.log(resultado);
// O resultado deve ser: Eu amo CapivaraJS. Capivara é demais.

Eventos

É possível escutar e emitir eventos. O CapivaraJS possui as funções $on e $emit que são responsáveis pelas respectivas ações.

capivara.$on

Para começar a escutar um evento, basta informar seu nome e uma função para ser executada quando seu evento for emitido. 

capivara.$on('meuEvento', function(){
    console.log('Uhaau! Meu evento foi executado');
});

capivara.$emit

A função $emit faz com que todos os ouvintes do seu evento seja executado. Exemplo: 

capivara.$emit('meuEvento');

Você pode enviar vários parâmetros no seu evento, para isso basta informa-los ao emitir um evento, logo após do seu nome. 
let meuParametro = 'CapivaraJS';
capivara.$emit('meuEvento', meuParametro);

Para receber os parâmetros enviados pela função $emit é feito da seguinte forma. 
capivara.$on('meuEvento', function(meuParametro){
    console.log(meuParametro)
});

Diretivas

cp-class

Com a diretiva cp-class você pode modificar dinâmicamente as classes dentro do seu componente. Exemplo: 

capivara.component('my-component', {
    template: `        
        <p cp-class="{ democlass: $ctrl.visible }">Um simples exemplo</p> 
        <button cp-click="$ctrl.teste()">Mudar cor</button>
    `,
    controller: function () {
        let $ctrl = this;
        
        $ctrl.$onInit = function(){
            $ctrl.visible = true;
        };
        $ctrl.teste = function(){
            $ctrl.visible = !$ctrl.visible;
        };
    }
});

cp-click

Você pode executar uma função do seu escopo. Exemplo: 

capivara.component('my-component', {
    template: `
        <button cp-click="$ctrl.minhaFuncao()">Clique-me</button>
    `,
    controller: function(){
        const $ctrl = this;

        $ctrl.minhaFuncao = function(){
            console.log('Minha função foi executada.');
        };
    }
});

cp-if

O cpIf te possibilita mostrar ou ocultar elementos de acordo com uma condição, diferente do cpShow os elementos são removidos do documento e contruidos dinamicamente. Exemplo: 

capivara.component('my-component', {
    template: `
        <h1 cp-if="$ctrl.mostrarTexto">Esse é um texto de exemplo.</h1>

        <button cp-click="$ctrl.alterar()">Clique para Mostrar</button>
    `,
    controller: function(){
        const $ctrl = this;

        $ctrl.mostrarTexto = false;

        $ctrl.alterar = function(){
            $ctrl.mostrarTexto = !$ctrl.mostrarTexto;
        };
    }
});

cp-else-if

O cpElseIf deve ser utilizado logo após a diretiva cpIf e outras diretivas CpElseIf, ela é visível quando o cpIf retornar falso e a condição da diretiva cpElseIf retorna verdadeiro. Exemplo: 

capivara.component('my-component', {
    template: `
        <h1 cp-if="$ctrl.contador == 2">H1 com IF tem a soma igual a [[$ctrl.contador]].</h1>
        <h1 cp-else-if="$ctrl.contador == 3">H1 com ELSE IF tem a soma igual a [[$ctrl.contador]].</h1>
        <h1 cp-else>H1 com ELSE tem a soma igual a [[$ctrl.contador]].</h1>

        <button cp-click="$ctrl.somar()">Clique para Somar</button>
    `,
    controller: function(){
        const $ctrl = this;

        $ctrl.contador = 1;

        $ctrl.somar = function(){
            $ctrl.contador +=1;
        };
    }
});

cp-else

O cpElse deve ser utilizado logo após a diretiva cpIf, ela é visível quando o cpIf retornar falso em sua condição. Exemplo: 

capivara.component('my-component', {
    template: `
        <h1 cp-if="$ctrl.mostrarTexto">Esse é um texto de exemplo.</h1>
        <h1 cp-else>Esse é um texto de exemplo.</h1>

    <button cp-click="$ctrl.alterar()">Clique para Mostrar</button>
    `,
    controller: function(){
        const $ctrl = this;

        $ctrl.mostrarTexto = false;

        $ctrl.alterar = function(){
            $ctrl.mostrarTexto = !$ctrl.mostrarTexto;
        };
    }
});

cp-init

O cpInit permite a execução de uma função quando um determinado elemento for renderizado. Exemplo: 

capivara.component('my-component', {
    template: `
        <h1 cp-init="$ctrl.inicializa('Esse é um exemplo')">String de teste</h1>
    `,
    controller: function(){
        const $ctrl = this;

        $ctrl.inicializa = function( value ){
            console.log('iniciou', value);
        };
    }
});

cp-model

Essa diretiva possibilita a interação do seu escopo com as input, quando usamos cp-model toda modificação de uma input é aplicada no escopo. Exemplo: 

capivara.component('my-component', {
    template: `
        <input cp-model="$ctrl.pessoa.nome"/>

        <button type="button" cp-click="$ctrl.test()">Clique-me</button>
    `,
    controller: function(){
        const $ctrl = this;

        $ctrl.pessoa = { nome: 'Mateus'};

        $ctrl.test = function(){
            console.log($ctrl.pessoa.nome);
        };
    }
});

cp-repeat

O cpRepeat te possibilita duplicar elementos html de acordo com um array do seu escopo. Exemplo: 

capivara.component('my-component', {
    template: `
        <ul>
            <li cp-repeat="pessoa in $ctrl.pessoas">
                [[pessoa.nome]]
            </li>
        </ul>
    `,
    controller: function(){
        const $ctrl = this;

        $ctrl.pessoas = [
            { nome: 'João' },
            { nome: 'Maria' }
        ];
    }
});

cp-show

O cpShow te possibilita mostrar ou ocultar elementos de acordo com uma condição. Exemplo: 

capivara.component('my-component', {
    template: `
        <h1 cp-show="$ctrl.mostrarTexto">Esse é um texto de exemplo.</h1>

        <button cp-click="$ctrl.alterar()">Clique para Mostrar</button>
    `,
    controller: function(){
        const $ctrl = this;

        $ctrl.mostrarTexto = false;

        $ctrl.alterar = function(){
            $ctrl.mostrarTexto = true;
        };
    }
});

cp-src

Utilizando cpSrc você pode adicionar ou remover dinâmicamente a utilização do src dentro do seu código. Exemplo: 

capivara.component('my-component', {
    template: `
        <img width="100px" cp-src="$ctrl.src" alt="">

        <button cp-click="$ctrl.click()">Clique para Mostrar</button>
    `,
    controller: function(){
        const $ctrl = this;

        $ctrl.src = 'https://avatars1.githubusercontent.com/u/33517395?s=200&v=4';

        $ctrl.click = () => {
            $ctrl.src = 'https://bit.ly/2pTjZnU';
        };
    }
});

cp-style

Com o cpStyle você pode modificar os style do seu componente dinâmicamente, através de outras diretivas como cp-click, inputs ou variáveis. Exemplo: 

capivara.component('my-component', {
    template: `        
        <h1 cp-style="{color: $ctrl.numberOne == 70 ? 'red' : 'green'}"> 
            [[ $ctrl.numberOne + $ctrl.numberTwo ]] 
        </h1>
        <p cp-style="{ background: 'red', padding: '10px'}">Exemplo 1</p>
        <p cp-style="{ background: $ctrl.blue, padding: '10px'}">Exemplo 2</p>
    `,
    controller: function () {
        let $ctrl = this;
        
        $ctrl.$onInit = function(){
            $ctrl.numberOne = 50;
            $ctrl.numberTwo = 80;
            $ctrl.blue = 'blue';
        };
    }
});
